# Jsonl to MDF

Script to convert json lines output from tracer to mdf file; used for visualization with Asammdf gui (https://canlogger.csselectronics.com/canedge-getting-started/ce2/log-file-tools/asammdf-gui/). Recommended to use uv python package manager.

# Usage

```bash
uv run ./jsonl_to_mdf.py ./trace.jsonl
```
